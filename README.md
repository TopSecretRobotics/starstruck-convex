# ConVEX skeleton project

Usage with docker:

```bash
docker build -t "docker-convex" -f Dockerfile .
docker run -v `pwd`:/build "docker-convex" sh -c 'make'
```

Usage with docker-compose:

```bash
# Start containers
docker-compose up -d
docker attach convex_project_1
# Stop containers
docker-compose stop
```
